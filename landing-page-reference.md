### 캘린더 디자인 사양

## 그런데 한 가지 분명한 사실

연초에 세운 계획을
혼자서 끝까지 끌고 가는 사람은
극소수입니다.

의지력의 문제가 아니에요.
시스템의 문제입니다.

✔️ 정해진 날짜가 없으면 "내일부터" 가 됩니다.
✔️ 함께 가는 동료가 없으면 "나만 안 해도 되겠지" 가 됩니다.
✔️ 도와주는 사람이 없으면 "나는 안 되나봐" 가 됩니다.

그래서, 혼자서는 절대 안 됩니다.


## 한 걸음씩 함께 나갈 수 있도록
## 제가 '앞선 발자국'이 되어드리겠습니다.

작년 10월,
'마지막 챌린지'라는 이름으로
1기를 모집했습니다.

3개월간 30개의 숏폼 튜토리얼을
정해진 날짜에 하나씩 업로드해 드리고,

여러분은 그것을 따라 만들고
4개 플랫폼에 차곡차곡 쌓는,

흔하지 않은 형태의 '실행 챌린지' 였습니다.

결과는,
제가 기대했던 것 이상이었습니다.

3개월간 단톡방은
하루도 조용한 날이 없었고,

서로 인증하고, 응원하고,
질문하고, 답해주는 그 풍경은

저에게도, 참여자분들에게도
평생 잊지 못할 경험이었습니다.

다만 한 가지 아쉬운 점은,
3개월간 단톡방에 켜켜이 쌓였던
그 모든 후기와 인사이트들이
방이 정리되면서 다 사라졌다는 거예요.

증거로 보여드릴 수 있는 게 지금은 없습니다.
(이 부분은 제가 1기를 운영하며 가장 크게 배운 점이기도 합니다.)

그래서 2기는,
처음부터 다시,
함께 차곡차곡 쌓아갑니다.


## 1기 → 2기, 무엇이 달라지나요?

크게 두 가지가 달라집니다.

**첫째, 주제가 더 좁고 더 깊어집니다.**

1기는 다양한 숏폼 제작 도구를 다뤘다면,
**2기는 '캡컷 PC 버전 마스터' 한 가지에 집중합니다.**

캡컷 모바일은 누구나 합니다.
하지만 PC 버전은 다릅니다.

PC 캡컷을 다룰 줄 알면
편집의 깊이와 속도가 완전히 달라지고,
**여러분이 만들 수 있는 콘텐츠의 퀄리티가 한 단계 점프합니다.**

3개월간 30개의 튜토리얼을 따라
직접 손으로 만들다 보면,
여러분은 자연스럽게
**캡컷 PC 마스터 레벨의 편집 감각**을
체득하게 됩니다.

이게 1석.

**둘째, 동시에 SNS도 채워집니다.**

튜토리얼대로 만든 영상 30개,
이걸 그냥 폴더에 묻어두지 않습니다.

여러분의 인스타 릴스에,
유튜브 쇼츠에,
틱톡과 네이버 클립에
하나씩 차곡차곡 업로드합니다.

3개월 후,
여러분의 SNS에는
**30개의 콘텐츠가 쌓여 있게 됩니다.**

이게 2석.

**1석2조.** 편집 감각 + 콘텐츠 자산.

이 두 가지를 동시에,
정해진 날짜에,
함께 가는 동료들과,
만들어 보자는 겁니다.


## 그래서, 여러분이 얻게 될 것들

3개월 후,
2026년 7월 30일,

이 챌린지를 완주하신다면
여러분은 이것들을 갖게 됩니다.

**1. 캡컷 PC 마스터 레벨의 편집 감각**
30개의 튜토리얼을 직접 손으로 만들어보면서 체득되는 편집 감각.
영상 강의만 백 번 봐서는 절대 얻을 수 없는,
'손이 기억하는' 편집 능력입니다.

**2. 30개의 숏폼 콘텐츠 포트폴리오**
폴더에만 잠자는 영상이 아닙니다.
여러분의 인스타·유튜브·틱톡·네이버 클립에
실제로 업로드된, 살아있는 자산.

**3. 알고리즘이 인식하는 '활성 계정'**
주기적으로 꾸준하게 콘텐츠가 업로드되는 계정,
알고리즘이 가장 좋아하는 계정입니다.
조회수가 서서히 오르며, 떡상 확률이 높아지는 계정으로 인식되기 시작합니다.

**4. 깨지지 않는 실행 루틴**
3개월간 정해진 날짜에 한 번도 빠지지 않고 업로드하는 경험.
이게 몸에 배면, 그 다음엔 어떤 도전을 해도
"나는 끝까지 해내는 사람"이 됩니다.

**5. 2026년 하반기 고속 성장의 발판**
기반이 탄탄한 4개의 플랫폼,
실행 루틴이 몸에 배인 상태,
어떤 콘텐츠가 먹히는지 데이터를 가진 채로
시작하는 2026년 하반기.

**그리고 마지막으로, 가장 중요한 것.**

**"나도 끝까지 해내는 사람이구나."**

이 자신감.

이 한 문장의 가치는,
직접 90일을 완주해본 사람만 압니다.

## 🖥️ 랜딩페이지 제작 요구사항

### 페이지 구조 (위에서 아래로)

1. **Hero 섹션**
   - 강력한 후크 헤드라인: "언제까지 '생각만' 하고 계실 건가요?"
   - 서브 카피: "2026년 상반기를 결산할, 마지막 3개월"
   - CTA 버튼: "9만원으로 신청하기" (스크롤시 sticky로 변환)
   - 우측 또는 하단에 D-Day 카운트다운 (얼리버드 마감까지)

2. **공감/문제 제기 섹션**
   - "다짐만 100번, 시작은 0번"
   - 3가지 체크 리스트 (정해진 날짜/함께 가는 동료/도와주는 사람)

3. **솔루션 섹션**
   - "앞선 발자국이 되어드립니다"
   - 1기 운영 경험 + 2기 진화 포인트

4. **차별화 섹션 (1석2조)**
   - 카드 2개 병렬 배치
   - "1석: 캡컷 PC 마스터 편집 감각"
   - "2석: 30개 SNS 콘텐츠 자산"

5. **얻게 될 것 섹션 (5가지)**
   - 카드 그리드 (모바일은 세로 1열, 데스크톱은 2~3열)
   - 각 카드: 짧은 제목 + 설명

6. **30회차 일정 시각화 섹션 ⭐**
   - 5월/6월/7월 캘린더 이미지 3장 가로 또는 세로 배치
   - "도망갈 수 없게 일정을 박아두었습니다" 카피
   - 모바일에서는 세로 스크롤로 자연스럽게

7. **제공 혜택 섹션 (6가지)**
   - 각 혜택을 카드로 시각화
   - "VOD 22강 (148,000원 상당)" 같은 가치 환산 강조
   - "🎁 [2기 신규]" 같은 뱃지 활용

8. **가격/참가비 섹션 (가장 중요!)**
   - 큰 가격 카드 2개 (얼리버드 9만원 vs 정상가 18만원)
   - 얼리버드 카드는 시각적으로 더 강조 (블루 보더, 살짝 큰 사이즈)
   - "9만원" 숫자는 매우 크게 (h1 사이즈)
   - **결제 CTA 버튼 ← 토스페이먼츠 연결**
   - 100명 한정 + 마감일 명시

9. **신청 방법 (3단계)**
   - 1️⃣ 결제 → 2️⃣ 신청폼 작성 → 3️⃣ 단톡방 입장
   - 각 단계마다 버튼/링크

10. **FAQ 섹션 (선택)**
    - 자주 묻는 질문 5~7개
    - 아코디언 UI

11. **클로징 + 최종 CTA**
    - "몰라서 안 되는 것이 아니라, 안 해서 안 되는 것이지요"

### 기능 요구사항

#### ⚡ 핵심: 토스페이먼츠 결제 연동
- **결제 수단**: 카드 (3개월 무이자 가능), 계좌이체
- **금액 분기 처리**:
  - 5월 4일(월) 00:00 ~ 5월 8일(금) 23:59:59 → 90,000원
  - 5월 9일(토) 00:00 이후 → 180,000원
- **자동 가격 전환**: JavaScript로 현재 시간 체크해 자동 표시
- **결제 후 처리**: 신청 완료 페이지 또는 단톡방 안내 페이지로 리다이렉트

#### 📊 D-Day 카운트다운
- 얼리버드 마감(2026-05-08 23:59:59 KST)까지의 남은 시간 실시간 표시
- 일/시/분/초 카운트
- 마감 후에는 "정상가 적용 중" 표시로 자동 전환

#### 📱 반응형
- 모바일 우선 설계 (네이버 블로그 모바일 트래픽이 큼)
- 380px 기준으로 깨짐 없이 표시
- 캘린더 이미지는 모바일에서 가로 100% 너비

#### 🎯 추가 기능 (있으면 좋음)
- 신청자 수 카운터 ("현재까지 X명 신청 / 100명 한정")
  - 임시로 정적 숫자도 가능, 백엔드 연결되면 동적
- 스크롤 시 진행률 인디케이터 (상단 얇은 바)
- CTA 버튼 sticky (모바일에서 하단 고정)
